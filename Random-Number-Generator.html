About the Random Number Generator

The ToolHub Random Number Generator produces single or multiple random numbers within a user-defined range. Choose whether you want integers or decimal values (with configurable precision), and optionally require the results to be unique. This utility is useful for lotteries, sampling, testing, games, classroom exercises, or any situation where you need reliable random values quickly.

Features:

  • Custom min / max range
  • Generate multiple numbers at once (count)
  • Option for unique results
  • Choose decimal precision (0–4)
  • Copy results to clipboard and download as CSV
  • Simple, mobile-friendly, and fast — no signup required

The generator uses secure random values where available and falls back to `Math.random()` when necessary. For unique integers, it ensures numbers are not repeated. When you need reproducible sequences (seeded generation), consider a dedicated seeded RNG; this tool focuses on quick ad-hoc generation.
